# Angular + Capacitor AI Plugin Skeleton

This Angular project is a **skeleton for testing my own custom Capacitor plugin**. The plugin uses the device camera to take a photo, analyzes it, and allows manipulation of colors â€” for example, changing black to white.  

The goal is to experiment with AI-powered image processing directly on mobile devices, useful for:

- Design apps  
- Clothing stores (preview color changes)  
- Just for fun and experimentation  

## Features

- Angular front-end  
- Capacitor integration for iOS and Android  
- Camera access via Capacitor  
- Custom AI plugin for color detection and manipulation  

## Getting Started

1. Clone the repository:  
git clone <https://github.com/Jaakko99/capacitor-camera.git>
cd <repo-folder>

2. Install Dependencies

npm install

3. Build it 
 
ng build

4. Sync and open capacitor

npx cap sync & npx cap open android / ios
